## Basic calculator

3.Functions and loops

## Calender and Reminder Application

I recently developed a Calendar & Reminder Application using Python that helps users efficiently manage their daily schedules, tasks and important events.

📅 What this app does:

1.Allows users to add, view and manage events
2.Sends timely reminders so nothing gets missed
3.Provides a simple and user-friendly interface
4.Helps improve productivity and time management

💡 This project is inspired by real-world calendar notification systems, which are designed to automate reminders and keep users organized by sending alerts for important tasks and deadlines.

🛠️ Technologies Used:
1.Python
2.Tkinter (for GUI)
3.JSON (for data storage)

🎯 Why I built this: Managing tasks manually can be overwhelming. This app simplifies scheduling by automating reminders and reducing the chances of missing important activities—just like modern reminder systems that enhance daily organization and efficiency .

📌 Key Learning: This project helped me understand GUI development, event handling, and how real-time reminder systems work.
✨ I’m continuously improving this app by adding new features!
